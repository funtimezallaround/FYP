\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

Computer vision analysis in aquatic environments presents unique challenges due to refraction, turbulence, and occlusion, which hinder the accuracy of standard motion tracking systems.
This project proposes a non-intrusive AI-based proof of concept system for reliable pixel-to-meter scaling using markers in the background.
The core contribution is a novel methodology for dynamic pixel-to-meter scaling using standard lane rope markers as calibration objects, addressing the limitations of fixed-camera setups in varying pool environments.
This progress report outlines the development of the modular marker prototypes and the architectural design of the computer vision pipeline. 
It concludes by defining the evaluation protocols and the remaining milestones required to validate the system's robustness in real-world scenarios
